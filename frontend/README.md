# Frontend

React + TypeScript + Vite + TailwindCSS chat UI.

## Features

- SaaS-style glassmorphism layout
- Responsive message thread with user/assistant/error bubbles
- Typing indicator animation
- Voice-to-text input (Web Speech API)
- Accessible keyboard interactions
- Smooth transitions, hover states, and auto-scroll behavior

## Commands

```bash
npm run dev
npm run build
npm run preview
```
